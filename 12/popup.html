1,2 Number Game

Rules:

1. Players take turns adding either 1 or 2 to the current sum.

2. The goal is to be the player to reach the target number exactly.

3. If a player goes over the target number, they lose.

4. The starting player is randomly determined.

How to Play Online Friend

1. Start the game and copy the game state to your clipboard.

2. Send the game state to your friend (via any messaging platform).

3. Your friend should paste the game state to start their turn.

4. After each turn, the current player should copy the new game state and send it to the other player.

5. The other player should paste the new state to see the updated game and take their turn.
